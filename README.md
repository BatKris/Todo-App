## Dependencies

#### The app requires:

* Node.js (v25.8.0 or higher), which you can download here [https://nodejs.org/en/download/current](https://nodejs.org/en/download/current)
* Vite build tool (v8.0.7 or higher), which comes with Node.js, so install that first. 
* An IDE (preferably VSCode), which you can download here [https://code.visualstudio.com/](https://code.visualstudio.com/)

## Set up

1. Download the zip file and extract the contents in a folder or clone the repository `https://github.com/BatKris/Todo-App.git`
2. Open a new terminal in the IDE and change the directory to the folder where the app is installed
3. Type `npm install` in the terminal, to download the required packages
4. After it's done, type in `npx vite dev` to run the app. You should see that the Vite server has started.
5. Then just press `o + Enter` on your keyboard and the web application should open in your browser
